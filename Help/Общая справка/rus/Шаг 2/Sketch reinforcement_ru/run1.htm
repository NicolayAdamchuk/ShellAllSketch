Запуск приложения и его результаты

Для выполнения приложения следует щелкнуть на кнопке Для спецификации.

Примечание: до выполнения команды файл проекта должен быть сохранен.



Если в проекте имеется арматура, то для нее создаются графические файлы формата png. Эти файлы размещаются в отдельной папке. По умолчанию папка создается в месте размещения файла проекта. При необходимости расположение папки можно изменить при помощи настроек в диалоговом окне.

В открывшемся диалоговом окне, можно активировать некоторые режимы:

  • Создать эскизы для всех стержней  - создаются эскизы для всех стержней проекта
  • Создать эскизы для выбранных стержней - создаются эскизы для стержней, выбранных на текущем виде. После выбора стержней следует нажать кнопку Готово
  • Создать эскизы для стержней текущего вида - создаются эскизы для стержней текущего вида
  • Создать эскизы для выбранного раздела - создаются эскизы для стержней определенного раздела (если разделы были назначены).

В случае выбора данной опции, становится доступным окно для выбора нужного раздела. Эскизы будут обновляться для выбранного раздела. Опция может быть полезна для больших проектов с большим числом арматурных стержней.


Кроме того, можно:  

  • включить флаг Показать длину крюков - для отображения длины крюков
  •  выбрать тип, размер и цвет шрифта
  • указать размер внешней границы рисунка.

Изменение данного параметра может быть полезным для улучшения компоновки эскиза и его читабельности.

Качество печати и визуализации эскиза зависит от резмера ячейки спецификации, где размещаются эскизы.

Рекомендуемый размер ячейки - 70х20 мм.

При увеличении данного параметра - размер рисунка с эскизом уменьшается.  


Вкладка Проектные формы.

На этой вкладке доступны опции для управления качеством эскиза:

  • Показать угол загиба. На эскизе будут показаны углы загиба стержня. Там где это уместно и возможно.
  • Прозрачный фон. При включении данного флага фон основания будет прозрачным (безцветным). В противном случае цвет будет белым.
  • Смещение от линии. Задается смещение текстовой надписи от линий эскиза.


Вкладка Стандартные формы.

       Группа параметров на данной вкладке позволяет настраивать эскизы арматуры для составных стержней. Эти параметры используются только для второго способа создания эскизов - на основе стандартных форм. Составные стержни формируются условно (в модели проекта изменения не производятся) с учетом назначенной максимальной длины стержня и нахлеста стержней. В тех случаях, когда длина стержня превышает максимальную производится разделение стержня на отдельные участки так, чтобы длина участка не превышала максимум.

Такие эскизы создаются для ограниченного числа стандартных форм. Опция работает для форм 1,11,12,13,21 (нормы BS8666:2005) и 1,2,3,4 (нормы ГОСТ 21-501). Например,
