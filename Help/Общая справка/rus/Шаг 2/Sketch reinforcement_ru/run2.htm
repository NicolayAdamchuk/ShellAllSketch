Запуск приложения и его результаты

Для выполнения приложения следует щелкнуть на кнопке Для чертежа.



Эскиз арматуры фомируется из линий, надписей и марки. Все позиции объединены в группу. Имя группы включает Id элемента армирования и имя вида, на котором был размещен экземпляр эскиза (при создании).




Пример эскиза



 В открывшемся диалоговом окне, можно выбрать следующие опции:


  • Марка для эскзиза  - при включении данного флага эскиз снабжается маркой.

Марка состоит из: марки пользователя, которая используется как префикс,  и дополнительной информации. После префикса указывается: количество стержней, диаметр арматуры и длина стержня (по оси).

  • Шрифт - указывается применяемый шрифт
  • Тип линии - указывается тип линии для вычерчивания эскиза
  • Коэффициент масштаба - указывается масштаб отображения эскиза
  • Выполнить выравнивание - при включении данного флага программа выполняет выравнивание эскиза относительно стержня в модели. При этом точку вставки эскиза можно указать приблизительно. Если флаг выключен, то эскиз будет размещен в точке указанной пользователем.
  • Удалять лишние эскизы при обновлении - при включении данного флага программа будет удалять лишние типы эскизов, которые могли остаться в проекте. Например, после удаления арматуры в модели.


Кнопка ОК.


       После щелчка на кнопке диалоговое окно закрывается и пользователь может выбрать для создания эскиза:

  • отдельный стержень
  • армирование по площади
  • армирование по траектории


       При выборе отдельного стержня создается один эскиз.

При выборе армирования по площади или траектории создаются эскизы для всех составляющих стержней.

Например,


       После выбора арматуры необходимо указать точку вставки эскиза для его размещения на виде.

       При выборе армирования по площади дополнительно открывается диалог, где следует выбрать направление и слой армирования. Эскизы будут созданы для этих параметров.

       


Кнопка Обновить вид.

         После щелчка обновляются эскизы текущего вида  с учетом изменений в армировании.

Кнопка Обновить все.

         После щелчка обновляются эскизы во всем проекте  с учетом изменений в армировании.

       Примечание: следует учитывать, что данная операция может потребовать некоторое время для выполнения.
