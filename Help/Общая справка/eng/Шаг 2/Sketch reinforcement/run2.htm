Running add-in and its results

To run the application you must click on the button For drawing. Note: project file should be saved before to run.

Reinforcement sketch is formed from lines, labels and tags. All objects are grouped together. The group name includes the Id of the reinforcement element and the name of the view on which the sketch instance was placed (at creation).




Sketch example




In the dialog box that opens, you can select next options:



  • Tag for sketch  - if the flag is switch on the sketch provided with the tag.

The tag consists of: the user's tag, which is used as a prefix, and additional information. After the prefix, the following is indicated: the number of bars of the given position, the diameter of the reinforcement and the length of the bar (along the axis).


  • Font- the applied font is selected
  • Line types - selects the line type for sketching
  • Scale coeff. -  specifies the sketch display scale
  • Perform aligment - when this checkbox is enabled, the program will align the sketch to the bar in the model. The sketch insertion point can be approximated. If the flag is off, the sketch will be placed at the point specified by the user.
  • Delete unnecessary sketches while updating - when this flag is enabled, the program will delete unnecessary types of sketches that may have remained in the project. For example, after removing reinforcement in the model.


Button ОК.


       After clicking on the button, the dialog box closes and the user can choose to create a sketch:

  • single rebar
  • area reinforcement
  • path reinforcement


       When you select a single rebar, creating a single sketch.

If selecting an area or path reinforcement - creates sketches for all component bars.

For example,



       After selecting the reinforcement, you must pick the insertion point of the sketch to place it in the view.

       When you select an area reinforcement, a dialog is additionally opened where you should select the direction and reinforcement layer. Sketches will be created for these options.


       


Button Update view.

         When clicked, the sketches of the current view are updated to reflect changes in the reinforcement.

Button Update all.

After clicking the sketches are updated in all project, taking into account changes in the reinforcement.

       Note: Please note that this operation may take some time to complete.
