Получение эскизов с использованием стандартных рисунков

В данном способе используется готовая библиотека рисунков со стандартными формами арматурных стержней. Для выполнения данного способа системное семейство формы стержня должно содержать ссылку на стандартный рисунок. Ссылка создается следующим образом:

- выберите арматурный стержень в проекте или форму системного семейства в Диспетчере проекта

- откройте Редактор семейства

- откройте Типоразмеры в семействе, нажав на кнопку

- для параметра Изображение формы щелкните на кнопке

 - в диалоговом окне Управление изображениями щелкните на кнопке Добавить

 - откройте папку с библиотекой рисунков, которая находится в «…Пользователи\Общие\Общие документы\Autodesk\Downloaded content\Ar-Cadia\SketchReinforcement»

 - откройте папку с подходящим стандартом. В настоящей версии доступны два стандарта BS8666-2005, Gost21-501 и EuroCod. Информация о стандартных рисунках находится ниже.

 - выберите подходящий стандартный рисунок, нажмите кнопку Добавить

 - теперь параметр Изображение формы содержит ссылку на нужный рисунок

 - нажмите кнопку ОК и выполните загрузку семейства в проект, нажав на кнопку (измененный файл семейства можно не сохранять)

 - данная процедура для текущей формы стержня выполняется один раз (для файла проекта или файла шаблона). Теперь каждый раз при применении данной формы стержня будет использоваться загруженный рисунок. Пример эскиза приводится ниже:
